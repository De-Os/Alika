# Alika
WinUI мессенджер для [vk.com](https://vk.com)

## Скачать
Скачать последнюю версию всегда можно в [канале с обновлениями](https://t.me/alika_vk)

## Описание и скриншоты
В функционал входят не только базовые возможности ВК, но и некоторые свои, например: [выгрузка диалогов](https://t.me/alika_faq/15) или [просмотр истории редактирования сообщения](https://t.me/alika_faq/14)

Разрабатывается в свободное время одним человеком и предоставляется "как есть".

Скриншоты можно найти в [канале с обновлениями](https://t.me/alika_vk)
## Проблемы
Из известных проблем:

* Могут наблюдаться вылеты на страницах без единого набора стикеров
* Если токен стал недействительным, приложение просто крашится ([решение](https://t.me/alika_faq/11))
* Для полной смены темы нужно полностью перезапустить приложение
* Кривая анимация на некоторых наборах стикеров
